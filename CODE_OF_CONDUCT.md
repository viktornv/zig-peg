# Code of Conduct

## Our Standard

We aim to keep this project welcoming, respectful, and collaborative.

Examples of expected behavior:

- Use constructive and respectful language.
- Give and receive feedback professionally.
- Focus discussions on technical outcomes and user value.

Examples of unacceptable behavior:

- Harassment, insults, or personal attacks.
- Discriminatory language or exclusionary behavior.
- Repeated disruptive communication after clear guidance.

## Scope

This applies to project spaces, including issues, pull requests, and discussions.

## Enforcement

Project maintainers may remove, edit, or reject contributions and interactions that violate this code.
